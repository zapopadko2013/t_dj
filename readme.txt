Описание Api тестовой задачи.

1. Запрос postphone на получение 4 значного кода по номеру телефона. Имеет тип POST.
   Маршрут запроса: http://127.0.0.1:8000/api/postphone/
   Тело запроса:
 {
    "phone": "1234567839",
    "name":"ffdh19"
  }
   Ответ:
  {
    "phone": "1234567839",
    "statusp": 1,
    "random": 4980,
    "text": "Телефон отсутствует в базе данных. Введите код:"
}

2. Запрос postcode на получение 6 значного пароля(инвайт-код) по номеру телефона.
Имеет тип POST.
   Маршрут запроса: http://127.0.0.1:8000/api/postcode/
   Тело запроса:
 {
    "phone": "1234567839",
    "code":"4980"
  }
   Ответ:
  {
    "phone": "1234567839",
    "statusp": 2,
    "random": "d_02GA",
    "text": "Телефон есть в базе данных. Введите пароль:"
}

3. Запрос postafterpassword для сохранения 6 значного пароля(инвайт-код) на завершние авторизации по номеру телефона.
Имеет тип POST.
   Маршрут запроса: http://127.0.0.1:8000/api/postafterpassword/
   Тело запроса:
 {
    "phone": "1234567839",
    "password":"d_02GA"
  }
   Ответ:
  {
    "phone": "1234567839",
    "statusp": 4,
    "text": "Пароль у телефона введён успешно!"
}

4. Запрос postupdatepassword для изменения пароля(инвайт-код)  по номеру телефона.
Имеет тип POST.
   Маршрут запроса: http://127.0.0.1:8000/api/postupdatepassword/
   Тело запроса:
 {
    "phone": "1234567839",
    "password":"d_02GA",
    "newpassword":"ffdhwe1."
  }
   Ответ:
  {
    "phone": "1234567839",
    "statusp": 4,
    "text": "Пароль у телефона заменён успешно!"
}

5. Запрос postpassword для проверки регистрации в базе данных по номеру телефона.
Имеет тип POST.
   Маршрут запроса: http://127.0.0.1:8000/api/postpassword/
   Тело запроса:
 {
    "phone": "1234567839",
    "password":"d_02GA"
  }
   Ответ:
  {
    "phone": "1234567839",
    "statusp": 3,
    "text": "Телефон есть в базе данных. Регистрация прошла успешно!"
}

6. Запрос profileuser для получение профиля пользователя по номеру телефона без пароля(инвайт-код).
Имеет тип GET.
   Маршрут запроса: http://127.0.0.1:8000/api/profileuser/?phone=1234567838
   
   Ответ:
  {
    "user": {
        "id": 1,
        "phone": 1234567838,
        "password": "ffdhwe1.",
        "name": "ffdh1"
    }
}

7. Запрос profileuserpassw для получение профиля пользователя по номеру телефона с паролем(инвайт-код).
Имеет тип GET.
   Маршрут запроса: http://127.0.0.1:8000/api/profileuserpassw/?phone=1234567838&password=ffdhwe1.
   
   Ответ:
  {
    "user": {
        "id": 1,
        "phone": 1234567838,
        "password": "ffdhwe1.",
        "name": "ffdh1"
    }
}

        {
